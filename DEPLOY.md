# How to Deploy & View Barbell Bahri on the Web

## 1. Enable GitHub Pages

1. Open **https://github.com/Bahri-adam/Barbell_Bahri**
2. Click **Settings** (top menu)
3. In the left sidebar, click **Pages** (under "Code and automation")
4. Under **Build and deployment**:
   - **Source**: Deploy from a branch
   - **Branch**: `main` (select from dropdown)
   - **Folder**: `/ (root)`
5. Click **Save**
6. Wait 1–2 minutes — GitHub will build the site

## 2. View Your App

Your app will be live at:

**https://bahri-adam.github.io/Barbell_Bahri/**

(Or with index: **https://bahri-adam.github.io/Barbell_Bahri/index.html**)

## 3. Add to iPhone Home Screen

1. Open **Safari** on your iPhone
2. Go to **https://bahri-adam.github.io/Barbell_Bahri/**
3. Tap the **Share** button (square with arrow)
4. Tap **Add to Home Screen**
5. Tap **Add**

## 4. Push Updates Later

When you make changes locally:

```bash
cd "c:\Users\abahri\OneDrive - Cherryland Electric Cooperative\Desktop\Powerbuilding program"
git add .
git commit -m "Describe your changes"
git push
```

GitHub Pages will rebuild automatically — refresh the site in 1–2 minutes.
