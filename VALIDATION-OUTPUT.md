# Engine Validation Report

**How to generate this file:**

1. Start a local server: `python -m http.server 3456`
2. Open http://localhost:3456/validation-test.html?autorun=1
3. Wait ~5 seconds for validation + stability to run automatically
4. Click **"Download VALIDATION-OUTPUT.md"**
5. Save to this location

Or run manually: open validation-test.html, click "Run Full Validation", then "Run Stability Tests", then "Download VALIDATION-OUTPUT.md".

*Replace this file with the downloaded report after running validation.*
